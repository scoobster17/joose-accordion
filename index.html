Joose Accordion Component

There are to types of accordion that can be created; the only difference is whether you can open one or multiple sections at once.

Please be aware this page is not yet functional due to wanting to set up node modules of a larger project, but can be used for reference of the accordion HTML to be used with the CSS and JS assets. To get this to work, you will also need assets from this Joose Component Library which contain utilities such as class manipulation. There are currently dependencies that rely on this, that are not fulfilled in this demo page.

No styling has been applied other than simple show/hide. You can utilise the existing HTML as I have, or you can add your own classes / selectors of your choice.

  • Single view accordion (collapses other content) var componentAccordion = new Accordion('#componentContainer', false); or <section id="accordionContainer" data-component="accordion" data-multiView="false">
  • Multi-view accordion (can view more than one portion at a time) var componentAccordion = new Accordion('#componentContainer', true); or <section id="accordionContainer" data-component="accordion" data-multiView="true">

The initialisation function will pick out any instances of accordions by searching for the data attribute automatically.
